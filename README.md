# NextCast.fm

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

NextCast.fm is an easy-to-use, flexible and open-source podcast manager for Android.

This project is a fork of [AntennaPod](https://github.com/AntennaPod/AntennaPod).

## Features

- Stream and download podcast episodes
- Playback speed control and skip silence
- Sleep timer with shake-to-reset
- Chapter support
- Queue management
- Subscription synchronization (gpodder.net, Nextcloud)
- OPML import/export
- Statistics and listening history
- Chromecast support
- And much more!

## Building

```bash
git clone https://github.com/karsentiz/NextCast.fm.git
cd NextCast.fm
./gradlew assembleDebug
```

The APK will be at `app/build/outputs/apk/free/debug/app-free-debug.apk`
